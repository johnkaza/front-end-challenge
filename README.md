## Description

Custom React Webpack implementation with typescript and React Boostrap for CSS framework and React Query for caching data and for tests I used Jest and Enzyme.

### Packages

- Webpack 4
- React
- React Bootstrap
- React Query (caching)
- Jest - Enzyme
- Typescript
- Axios

## Installation

```
cd app

npm install
```

## Development

```
npm run start
```

## Production

```
npm run build
```

## Test

```
npm run test
```
